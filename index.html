Sobre los Tests de PreparaMAC

Preparación para las oposiciones de Auxiliar Administrativo -Informática-

Bienvenido a la Aplicación de Test de Informática de preparaMAC. Esta aplicación te ayudará a preparar las pruebas de informática para las oposiciones de Auxiliar Administrativo.

Aquí encontrarás test basados en preguntas de convocatorias anteriores, organizados por temas y con un sistema de autoevaluación.

¿Cómo usar la aplicación?

✅ Selecciona un tema o test desde el menú principal.

✅ Responde a las preguntas y revisa las soluciones.

✅ Practica tantas veces como necesites.
