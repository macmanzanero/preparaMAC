Gráficos Otros Elementos Gestión y Análisis

Esquema Interactivo: Excel, Gestión de Datos

Repase de forma concisa y estructurada los conceptos clave de la gestión y análisis de datos en Microsoft Excel para su examen de informática.

1. Inserción de Gráficos: Visualización de Datos

1.1 Tipos de Gráficos ▶

Columnas/Barras: Comparar valores entre categorías.

Líneas: Mostrar tendencias a lo largo del tiempo.

Circulares/Anillos: Representar proporciones de un total.

Dispersión: Mostrar relación entre dos variables numéricas.

Área: Tendencias a lo largo del tiempo, enfatizando el volumen.

Combinados: Mezclar tipos de gráficos para diferentes series de datos.

1.2 Creación de Gráficos (Métodos) ▶

Automática (Análisis Rápido): Selección de datos y uso de "Análisis Rápido" (Ctrl + Q) para sugerencias.

Recomendados: Excel sugiere gráficos basados en los datos seleccionados.

Desde Grupo Gráficos: Pestaña "Insertar" > Grupo "Gráficos" > Selección manual del tipo.

1.3 Personalización de un Gráfico ▶

Elementos: Título del gráfico, títulos de ejes, leyenda, etiquetas de datos, tabla de datos.

Estilos y Colores: Aplicar temas predefinidos o personalizar colores de series.

Diseño Rápido: Plantillas de diseño que combinan elementos y formato.

Cambiar Tipo: Modificar el tipo de gráfico tras su creación.

1.4 Borrar un Gráfico ▶

Seleccionar el gráfico y pulsar Supr (Delete).

2. Inserción de Otros Elementos en la Hoja

2.1 Imágenes ▶

Insertar desde archivo local o desde fuentes online.

Propiedades: Tamaño, posición, ajuste de texto.

2.2 Formas ▶

Elementos gráficos predefinidos (rectángulos, círculos, flechas, llamadas).

Uso: Resaltar información, crear diagramas sencillos.

2.3 Elemento Gráfico SmartArt ▶

Representaciones visuales de información e ideas.

Tipos: Listas, procesos, ciclos, jerarquías, relaciones, matrices, pirámides, imágenes.

Uso: Organizar y comunicar información de forma efectiva.

2.4 Otros Elementos ▶

Cuadros de Texto: Insertar texto independiente de las celdas.

Símbolos: Caracteres especiales no disponibles en el teclado.

Ecuaciones: Fórmulas matemáticas complejas.

3. Gestión y Análisis de Datos

3.1 Buscar Datos ▶

Buscar (Ctrl + B): Localizar texto o números específicos.

Reemplazar (Ctrl + L): Buscar y sustituir datos.

3.2 Tablas (o Bases de Datos en Excel) ▶

Concepto: Rango de celdas organizado con encabezados (campos) y filas (registros).

Ventajas: Facilita ordenación, filtrado, cálculo de totales, y expansión automática.

Creación: Seleccionar rango y "Dar formato como tabla" (Pestaña Inicio).

3.3 Ordenar Datos ▶

Ascendente/Descendente: Ordenar por uno o varios campos.

Por Múltiples Columnas: Definir niveles de ordenación.

Orden Personalizado: Definir listas personalizadas para ordenar (ej. días de la semana).

Uso: Organizar la información para un mejor análisis o visualización.

3.4 Filtrar Datos ▶
Filtro Automático ▶

Mostrar solo las filas que cumplen un criterio.

Filtro de Texto, Número, Fecha.

Filtros personalizados (mayor que, entre, etc.).

Filtro Avanzado ▶

Criterios complejos en un rango separado. Permite extraer datos a otra ubicación.

Uso: Aislar subconjuntos de datos para análisis específicos.

3.5 Buscar Objetivo ▶

Concepto: Herramienta de análisis "Y si..." (What-If Analysis).

Uso: Determinar qué valor de entrada se necesita para obtener un resultado deseado en una fórmula.

Ejemplo: ¿Qué ventas necesito para alcanzar un beneficio de X?

3.6 Auditoría de Fórmulas ▶

Rastrear Precedentes: Mostrar las celdas que afectan a una fórmula.

Rastrear Dependientes: Mostrar las celdas afectadas por una fórmula.

Mostrar Fórmulas: Visualizar las fórmulas en lugar de los resultados.

Comprobación de Errores: Identificar y depurar errores en las fórmulas.

Uso: Entender y depurar la lógica de los cálculos en hojas complejas.

3.7 Validación de Datos ▶

Concepto: Establecer reglas para restringir el tipo o los valores de datos que se pueden introducir en una celda.

Tipos de Criterios: Número entero, decimal, lista (desplegable), fecha, hora, longitud de texto, personalizada (con fórmula).

Mensajes: De entrada (ayuda) y de error (al no cumplir la regla).

Uso: Garantizar la calidad y coherencia de los datos.

3.8 Subtotales ▶

Concepto: Agrupar datos y realizar cálculos de resumen (suma, promedio, recuento) automáticamente.

Requisito: Los datos deben estar previamente ordenados por la columna de agrupación.

Uso: Resumir grandes conjuntos de datos de forma jerárquica.

3.9 Tablas Dinámicas (PivotTables) ▶

Concepto: Herramienta interactiva para resumir, analizar, explorar y presentar grandes volúmenes de datos.

Componentes: Campos de filas, columnas, valores (cálculos) y filtros.

Ventajas: Flexibilidad para reorganizar datos, análisis rápido sin modificar la fuente original.

Uso: Obtener información clave de forma dinámica (ej. ventas por región y producto).

3.10 Consolidar Datos ▶

Concepto: Combinar datos de diferentes rangos de celdas o hojas de cálculo en una única ubicación.

Métodos:

Por Posición: Los datos están en el mismo orden y ubicación en cada rango.

Por Categoría: Los datos tienen etiquetas de fila y/o columna comunes.

Funciones: Suma, promedio, recuento, etc.

Uso: Unificar información de múltiples fuentes (ej. presupuestos trimestrales de diferentes departamentos).
