Esquema Interactivo: Excel, Entorno de Trabajo

Repase de forma concisa y estructurada los conceptos clave de Microsoft Excel para su examen de informática.

1. Hojas de Cálculo. Microsoft Excel: El Entorno de Trabajo

1.1 Introducción ▶

Hoja de Cálculo: Software para manejar datos en tablas (filas/columnas). Permite calcular, graficar y organizar información.

Excel: La hoja de cálculo de Microsoft. Usos: contabilidad, planificación, análisis de datos, creación de gráficos.

1.2 Gestión del Programa (Inicio/Cierre) ▶

Abrir: Desde el menú del SO o abriendo un archivo .xlsx.

Guardar (Ctrl + G): Para no perder cambios.

Guardar como: Para guardar con otro nombre o en otra ubicación.

Cerrar: Finalizar la sesión con el archivo o la aplicación.

1.3 Componentes del Entorno (GUI) ▶

Cinta de Opciones (Ribbon): Barra superior con pestañas (Inicio, Insertar) que agrupan herramientas.

Barra de Fórmulas: Muestra y permite editar el contenido real de la celda activa (valor o fórmula).

Cuadro de Nombres: Muestra la referencia de la celda seleccionada (ej. A1). Útil para navegar.

Área de Trabajo: La cuadrícula principal de filas y columnas donde se introducen los datos.

Barra de Estado: Abajo, muestra información contextual (ej. suma de celdas seleccionadas).

1.4 Sistema de Ayuda ▶

Acceso rápido a la documentación y funciones de búsqueda integradas.

1.5 Navegación y Desplazamiento ▶

Ratón: Clic para seleccionar, arrastrar barras de desplazamiento.

Teclado: Flechas (celda a celda), Enter (abajo), Tab (derecha), Ctrl + Flechas (saltos rápidos).

2. Libros, Hojas y Celdas: Estructura

2.1 Libros (Workbooks) ▶

El archivo principal de Excel (.xlsx). Contiene una o varias hojas.

Operaciones: Abrir, guardar, cerrar.

2.2 Hojas (Worksheets) y Celdas (Cells) ▶

Hoja: Cada "página" dentro de un libro. Organiza datos relacionados. Se pueden añadir, eliminar, renombrar.

Celda: La unidad mínima de datos (intersección fila/columna). Tiene una referencia única (ej. B4).

Filas: Horizontales, numeradas ($1, 2, 3...$).

Columnas: Verticales, con letras ($A, B, C...$).

Referencia de Celda: La dirección única de una celda (ej. D5). Esencial para fórmulas.

3. Introducción y Edición de Datos

3.1 Introducción de Datos ▶

Tipos: Texto, números, fechas/horas, fórmulas (siempre con =).

Método: Seleccionar celda y escribir. Confirmar con Enter o Tab.

3.2 Modificación y Eliminación ▶

Editar: Doble clic, F2, o en la Barra de Fórmulas.

Borrar: Supr (Delete) en la celda/rango.

3.3 Notas y Comentarios ▶

Notas: Aclaraciones personales sobre una celda.

Comentarios: Para discusiones colaborativas en celdas.

3.4 Selección ▶

Celda: Clic.

Rango: Clic y arrastrar.

Fila/Columna: Clic en el encabezado.

Toda la hoja: Clic en el botón superior izquierdo.

3.5 Rangos de Celdas ▶

Grupo de celdas adyacentes. Se define por sus esquinas (ej. A1:C5).

3.6 Edición Básica ▶

Copiar/Pegar (Ctrl + C/V): Duplicar.

Cortar/Pegar (Ctrl + X/V): Mover.

Relleno Automático: Arrastrar la esquina inferior derecha para replicar o generar series.

3.7 Inserción y Borrado (Filas/Columnas) ▶

Añadir o quitar filas/columnas completas. Afecta la estructura y las referencias de fórmulas.

3.8 Ajuste Dimensional ▶

Manual: Arrastrar bordes de encabezados.

Autoajuste: Doble clic en el borde para ajustar al contenido.

4. Formatos: Presentación y Legibilidad

4.1 Formato de Fuente ▶

Tipo, tamaño, negrita, cursiva, subrayado, color. Mejora la apariencia del texto.

4.2 Paleta de Herramientas Formato ▶

En la pestaña "Inicio" (grupos Fuente, Alineación, Número).

4.3 Alineación ▶

Horizontal: Izquierda, Centro, Derecha.

Vertical: Superior, Medio, Inferior.

Ajustar Texto: Muestra texto largo en varias líneas dentro de la celda.

Combinar y Centrar: Une celdas para un título.

4.4 Bordes y Relleno (Tramas) ▶

Bordes: Líneas para delimitar celdas.

Relleno: Color de fondo de la celda.

4.5 Formato Condicional ▶

Cambia el formato de una celda automáticamente si cumple una condición (ej. notas < 5 en rojo).

4.6 Formato de Datos ▶

Define cómo se muestran números (decimales, miles), moneda, porcentajes, fechas/horas.

5. Previsualización e Impresión

5.1 Vista Preliminar ▶

Permite ver cómo quedará la hoja en papel antes de imprimir. Evita errores.

5.2 Configuración e Impresión ▶

Configuración de Página: Márgenes, orientación (vertical/horizontal), tamaño de papel.

Área de Impresión: Seleccionar solo una parte de la hoja para imprimir.

Imprimir: Enviar a la impresora.

6. Fórmulas y Funciones: La Potencia de Cálculo

6.1 Fórmulas ▶

Expresiones que realizan cálculos. Siempre empiezan con =.

Muestran un resultado en la celda, pero la expresión se ve en la Barra de Fórmulas.

6.2 Elementos de una Fórmula ▶

Operadores (+, -, *, /, ^).

Referencias de celda (ej. A1).

Valores directos (ej. 10).

Funciones (ej. SUMA).

6.3 Fórmulas Sencillas ▶

Ej: =A1+B1 (suma A1 y B1).

Ej: =C5*10 (multiplica C5 por 10).

6.4 Fórmulas con Referencias ▶

Usan direcciones de celda. Si los valores de las celdas cambian, el resultado se actualiza solo.

6.5 Tipos de Referencia (Clave) ▶
Relativas (ej. A1) ▶

Comportamiento: Cambian automáticamente al copiar la fórmula.

Uso: Para aplicar la misma lógica a datos adyacentes.

Ej: =A1+B1 en C1, al copiar a C2, se vuelve =A2+B2.

Absolutas (ej. $A$1) ▶

Comportamiento: No cambian al copiar la fórmula. Se fijan con $.

Uso: Cuando una celda debe ser siempre la misma referencia (ej. un valor fijo para todos los cálculos).

Ej: =$A$1+B1 en C1, al copiar a C2, se mantiene =$A$1+B2.

Mixtas (ej. A$1, $A1) ▶

Comportamiento: Una parte (fila o columna) es fija, la otra cambia.

A$1: Fila fija, columna cambia.

$A1: Columna fija, fila cambia.

Uso: En tablas de doble entrada o cálculos específicos.

6.6 Fórmulas con Funciones ▶

Las funciones son cálculos predefinidos que simplifican las fórmulas.

Ej: =SUMA(A1:A5) (suma el rango).

Ej: =PROMEDIO(B1:B10) (calcula la media).

6.7 Funciones (Ejemplos) ▶

Matemáticas/Estadísticas: SUMA, PROMEDIO, MAX, MIN, CONTAR.

Lógicas: SI (evalúa una condición).

Texto: CONCATENAR.

Fecha/Hora: HOY, AHORA.

7. Vínculos (Referencias entre Hojas y Libros)

7.1 Referencias Internas ▶

Misma Hoja: =A1

Otra Hoja del Mismo Libro: =NombreHoja!A1 (ej. =Hoja2!A1).

7.2 Referencias Externas ▶

Conexión a celdas en otros archivos de Excel.

Sintaxis: ='[NombreArchivo.xlsx]NombreHoja'!A1

Uso: Consolidar datos de varios informes.
